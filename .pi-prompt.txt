You are improving a personal portfolio site (serhiifotex.dev). The site is a Vite + React + TypeScript + Tailwind CSS v4 SPA deployed on Vercel.

## Project Structure
- src/App.tsx — main app with profile section + hash routing + blog
- src/components/BlogSection.tsx — blog post list with tag filtering
- src/components/LinkButton.tsx — glowing link card
- src/components/PostPage.tsx — single blog post view
- src/components/MarkdownContent.tsx — custom markdown renderer
- src/content/posts.ts — loads .md files from src/posts/ with frontmatter
- src/posts/ — markdown blog posts (hello-word.md, recon-search.md)
- src/styles/globals.css — Tailwind + custom styles (wabi-sabi dark theme)
- index.html — entry HTML

## Tech Stack
- Vite 7, React 19, TypeScript 5.9, Tailwind CSS v4
- Phosphor Icons, react-toastify, react-tooltip
- @vercel/analytics
